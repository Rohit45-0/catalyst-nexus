## 🚨 Important Notes

### Instagram API Limitations:
- City data available after **24-48 hours**
- Need **1000+ impressions** for city breakdown
- API calls limited (200/hour)
- Access token expires (refresh needed)

### Data Collection Tips:
1. **Run for 48 hours** per post minimum
2. **Publish 10-20 posts** to build dataset
3. **Vary content** to test different spread patterns
4. **Check data quality** with `view_gnn_data.py`
5. **Export regularly** to avoid data loss

### GNN Training Tips:
1. **Start simple** (2-layer GCN)
2. **Use temporal features** (time since publish)
3. **Normalize features** (StandardScaler)
4. **Split data** (80% train, 20% test)
5. **Cross-validate** across campaigns
